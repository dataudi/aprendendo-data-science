# Como contribuir

Clone o repositório para sua estação de trabalho:

```
git clone git@github.com:dataudi/aprendendo-data-science.git
```

Edite o arquivo README.md:
* Localize o idioma, se o conteúdo é gratuito ou pago, e o tipo/formato do recurso
* Acrescente o link do novo recurso ao final da lista
* Dê um título ao recurso
* Para acrescenter um link utilize o formato (título)[url]
* Bom aprendizado!
